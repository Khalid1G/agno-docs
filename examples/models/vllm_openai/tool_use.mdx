title: Agent with Tools
---

Make sure to start the vLLM server first:

```shell
vllm serve Qwen/Qwen3-8B \
    --enable-auto-tool-choice \
    --tool-call-parser hermes \
    --dtype float16 \
    --max-model-len 8192 \
    --gpu-memory-utilization 0.9 \
    --trust-remote-code
```

<Note>
For more information about vLLM tool use, please refer to [our guide on Function Calling](https://qwen.readthedocs.io/en/latest/framework/function_call.html#vllm).
</Note>

## Code

```python cookbook/models/vllm/tool_use.py
from agno.agent import Agent
from agno.models.vllm import vLLM
from agno.tools.duckduckgo import DuckDuckGoTools

agent = Agent(
    model=vLLM(
        id="Qwen/Qwen3-8B", top_k=20, enable_thinking=False
    ),
    tools=[DuckDuckGoTools()],
    show_tool_calls=True,
    markdown=True,
)
agent.print_response("Whats happening in France?", stream=True)
```
